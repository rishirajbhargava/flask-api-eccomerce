Welcome to Ecommerce Api

API Description

This is a simple REST API for Ecommerce website/products api. It is built with Flask and Sqlite database. It has the following features:

  • Get all users stored in database
  • Get a single user by Id stored in database
  • Get all orders stored in database
  • Get a single order stored in database
  • Add a new user to database
  • Add a new order to database
    •

    Tech & Libaries used:

    Flask Jinja2 Flask-SQLAlchemy SQLAlchemy Postman VS Code Git & Github HTML CSS Google Fonts Font Awesome

    API Endpoints

  • Get all users :
    Route - /users | Method - GET
  • Get single user:
    Route - /users/{id} | Method - GET
  • Get all orders :
    Route - /orders | Method - GET
  • Get a single order:
    Route - /orders/{id} | Method - GET
  • Add a order :
    Route - /orders | Method - POST
  • Add a user :
    Route - /users | Method - POST

    • User Model

  • id : Unique Id of the user (generated automatically)
  • name: Name of the user (can be anything)
  • email: Email of the user.

    • Order Model

  • id : Unique Id of the Order (generated automatically)
  • user_id: Unique id of user stored in database.
  • product_name: Name of the Product.
  • product_name: Name of the Product.
  • quantity: Quantity of the Product.
  • total_price: Total Price of the order.
  • created_at: Date of creation of the order.(generated automatically)
